### Setup

Steps:

-   install dependencies

-   create config.json -> src/config directory

-   add base environment config -> config.json

-   run server

```javascript

{
    "NODE_PORT": 8080
}

```

```shell

$ npm i

$ cd src/config & touch config.json

$ npm run dev

```
